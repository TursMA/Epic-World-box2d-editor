Epic World editor

A minimalist Box2D world editor that just wants to provide enough functionalities.
We don't need to reinvent the feature to edit bodies, this tool is designed to be used in conjuction with Physics Body Editor!

Essential functionalities completed, including import/export, moving, scaling, undo/redo. It's now ready to be used!

Download

  • For Windows, binary distribution of version 1.9 can be downloaded here.
  • For Linux, running the code is very easy, just download the source code, install PyQt5 (package python3-pyqt5 on Ubuntu) and run python3 epicworld.py.
